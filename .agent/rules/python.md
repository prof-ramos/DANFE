* Versão mínima: Python 3.10+
* Gerenciador de dependências: `uv` (não pip, não poetry)
* Ambiente virtual: `uv venv`
* Padrões de código: PEP 8, type hints completos, docstrings em formato Google
* Testes: pytest com cobertura mínima de 80%
